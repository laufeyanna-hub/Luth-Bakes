## 🚀 Tentang Proyek
Luthé Bakes adalah bisnis homemade bakery dari Metro, Lampung yang fokus pada:
- 🍫 Brownies harian (ready stock)
- 🎂 Bento cake custom (pre-order)
- 🎁 Hampers kue kering (seasonal: Lebaran/Natal)
